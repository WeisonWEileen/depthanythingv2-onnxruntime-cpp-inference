####  platform
- ubuntu 22.04
- onnxruntime 18.0
- CUDA 12.2
- cudnn 8.9.x.x

### to reproduce 
- set your onnxruntime library path in CMakeLists.txt
- run ```build.sh```
- run ```./build/src/main``` in your project's root directory
